# Adani Enterprises Stock Analysis (2021–2026)

## Project Overview
This project analyzes Adani Enterprises stock performance using NSE historical data from 2021 to 2026.

The analysis was performed using Python (Pandas) and visualized using Tableau to identify stock price trends, turnover patterns, and key business insights.

## Tools Used
- Python
- Pandas
- Tableau
- Microsoft Word

## Dashboard KPIs
- Highest Closing Price: 4,165
- Highest Average Turnover: ₹685.17 Cr
- Highest Average Price: 4,125

## Key Insights
- Significant stock growth during 2022.
- Peak trading turnover observed during 2023.
- Stock price reached its highest average level during 2024.
- Market volatility was followed by recovery and stabilization.
- Business events had a visible impact on stock performance.
